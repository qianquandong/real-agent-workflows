# Obsidian · 首页驾驶舱

> 三个信息源加一个邮箱，每天早上自动汇成一页早报。你要做的只是边吃早饭边打勾。

## 这条工作流长什么样

| | |
|---|---|
| **触发** | 每天早上定时（作者设在 06:11），起床前跑完 |
| **步骤** | 三源抓料（AI 热点 / GitHub trending / 非 AI 新闻）→ 对台账去重 → Gmail MCP 把过夜邮件分三类 → 组装首页 |
| **产出** | Obsidian 首页两节：「待勾选的新素材」（每条带复选框）+「Gmail Action」（每条带直达链接） |
| **人工** | 早餐时间扫一眼，想留的素材打个勾 |

## 设计要点（为什么长这样）

- **勾选是唯一的入库闸门**。这条工作流只列素材、绝不入库——入库由晚上的任务处理勾中的条目。人不看过的东西不进知识库，库才不会变垃圾场。
- **只动自己的两节**。首页由多个定时任务共写（早报、短视频清单、待办各归各的任务管），每个任务写到下一个 `## ` 标题为止。这条规矩让 N 个任务共用一页而互不覆盖。
- **Gmail 连不上就明说**。宁可写「今日 Gmail 未连上」，绝不编一封不存在的邮件。

## 怎么装

1. 把 [`SKILL.md`](./SKILL.md) 复制到 `~/.claude/skills/obsidian-morning-cockpit/SKILL.md`
2. 替换三个占位符：库路径、首页文件名、你的关注领域
3. Gmail 分类需要连上 Gmail MCP（Claude 的 Gmail 连接器）；不连也能跑，只是少邮件那一节
4. 手动跑一次看首页效果，再挂定时
