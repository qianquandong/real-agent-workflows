# Obsidian · 首页驾驶舱

> 三个信息源加一个邮箱，每天早上自动汇成一页早报。你要做的只是边吃早饭边打勾。

## 这条工作流长什么样

| | |
|---|---|
| **触发** | 每天早上定时（作者设在 06:11），起床前跑完 |
| **步骤** | 三源抓料（AI 热点 / GitHub trending / 非 AI 新闻）→ 对台账去重 → 邮箱连接器把过夜邮件分三类 → 组装首页 |
| **产出** | Obsidian 首页两节：「待勾选的新素材」（每条带复选框）+「邮箱 Action」（每条带直达链接） |
| **人工** | 早餐时间扫一眼，想留的素材打个勾 |

## 设计要点（为什么长这样）

- **勾选是唯一的入库闸门**。这条工作流只列素材、绝不入库——入库由晚上的任务处理勾中的条目。人不看过的东西不进知识库，库才不会变垃圾场。
- **只动自己的两节**。首页由多个定时任务共写（早报、短视频清单、待办各归各的任务管），每个任务写到下一个 `## ` 标题为止。这条规矩让 N 个任务共用一页而互不覆盖。
- **邮箱连不上就明说**。宁可写「今日邮箱未连上」，绝不编一封不存在的邮件。
- **内容只是作者在意的东西**。信息源随你换，每天要盯的别的事（打卡、家庭待办、学校通知）都可以往这页早报里加一节——结构通用，内容是你的。

## 怎么装

1. Claude Code 用户：把 [`SKILL.md`](./SKILL.md) 复制到 `~/.claude/skills/obsidian-morning-cockpit/SKILL.md`；其他 agent 工具见[根 README 的通用说明](../README.md#怎么用)
2. 替换三个占位符：库路径、首页文件名、你的关注领域
3. 邮箱分类需要一个邮箱连接器/MCP（作者用 Gmail 的；其他邮箱换成对应工具）；不连也能跑，只是少邮件那一节
4. 手动跑一次看首页效果，再挂定时
